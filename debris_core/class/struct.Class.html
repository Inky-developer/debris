pub struct Class {
    pub kind: ClassKind,
    pub properties: RefCell<ObjectProperties>,
}
Expand description

A class combines ClassKind and corresponding methods. Due to the way classes get initialized, the methods have to be stored in a RefCell. For more info see crate::compile_context::TypeContext and debris_derive::object.
