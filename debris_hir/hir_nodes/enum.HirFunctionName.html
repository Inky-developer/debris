pub enum HirFunctionName {
    Ident(SpannedIdentifier),
    Unnamed {
        file: CodeId,
        pos: Span,
    },
}
Expand description

Anonymous functions don’t have an explicit identifier, So if the function is unnamed, the hir builder will try to infer it.
