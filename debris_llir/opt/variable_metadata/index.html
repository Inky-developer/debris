Module debris_llir::opt::variable_metadata

source ·
Expand description

This module contains structs which can hold metadata for variables. This includes so called ValueHints which allow to deduce the set of possible runtime values a variable can have. This module also contains variable usage data which contain information about the amount of reads and writes a specific variable gets
